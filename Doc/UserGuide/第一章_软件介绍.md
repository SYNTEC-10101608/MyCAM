# 第一章 软件介绍

## 本章概要

本章介绍 MyCAM 五轴激光切割软件的基本信息、主要功能、系统需求和启动流程。

## 1.1 软件概述

**MyCAM** 是一款专门用于五轴激光切割加工设计的 CAD/CAM 集成软件系统。该软件结合了计算机辅助设计（CAD）和计算机辅助制造（CAM）功能，提供从 3D 模型导入到 NC 程序输出的完整解决方案。

## 1.2 软件用途与应用场景

### 主要功能模块

**MyCAM** 软件主要提供以下核心功能：

**1. 3D 数模导入**
- 3D 模型导入（支持 STEP、IGES 格式）

**1. 3D 零件设计**
- 模型缝补与检查
- 特征元素创建
- 坐标系转换与调整

**2. 3D 路径规划**
- 路径生成
- 路径排序
- 路径工艺（起切点、加工方向、引线、过切、微连）
- 姿态调整与优化
- 蛙跳规划
- 路径编辑
- 三点校正
- NC 程序输出

**3. 3D 仿真功能**
- 实时加工仿真
- 碰撞检测

**4. CNC 数控**
- All-in-one CAD/CAM + CNC 操作界面

## 1.3 系统需求

### 1.3.1 硬件需求

| 硬件 | 最低需求 | 推荐配置 |
|------|----------|----------|
| **处理器** | Intel Core i5 或同级别 | Intel Core i7 或更高 |
| **内存** | 8 GB RAM | 16 GB RAM 或更高 |
| **显卡** | 支持 OpenGL 3.3 以上 | 专业显卡，支持 OpenGL 4.5 |
| **硬盘空间** | 5 GB 可用空间 | 10 GB 以上（含项目文件空间）|
| **屏幕分辨率** | 1920 × 1080 | 2560 × 1440 或更高 |
| **网络** | 可选（用于 VNC 连接）| 100 Mbps 以上有线网络 |

### 1.3.2 软件需求

| 项目 | 需求 |
|------|------|
| **操作系统** | Windows 11（64 位版）|
| **.NET Framework** | .NET Framework 4.8 或更高版本 |
| **OpenCascade** | 已集成到软件（软件内置）|
| **VNC Viewer** | TightVNC 或兼容软件（用于远程连接功能）|

## 1.4 授权与启动说明

### 1.4.1 软件启动

- 双击执行 `MyCAM.exe`
- 系统将自动进行授权验证

### 1.4.2 授权机制

MyCAM 采用授权验证机制，软件启动时会自动进行授权检查。

**授权检查流程：**

```
软件启动
    ↓
授权验证
    ↓
验证授权状态
    ↓
[授权有效] → 软件正常启动    |    [授权无效] → 显示错误信息并自动关闭
```

> **⚠️ 注意事项：**
> - 如出现授权错误，请联系系统管理员或软件供应商

> **📸 截图建议 1-3：启动界面 - 软件成功启动后的初始界面，显示各功能区已正常载入**  
> **对应文件名称：** `ch01_03_启动界面.png`  
> **重要性：** ⭐⭐ 建议

### 1.4.3 常见启动问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 软件无法启动 | .NET Framework 未正确安装 | 安装 .NET Framework 4.8 |
| Viewer 初始化错误 | 未安装 VC++ 可再发行组件包 | 安装 VC++ 可再发行组件包 |
| 授权验证失败 | 授权过期或无效 | 联系软件供应商更新授权 |
| 机床数据载入失败 | 配置文件丢失 | 检查 MachineData.mac 文件 |
