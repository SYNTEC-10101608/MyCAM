## 1.1 软件概述

**MyCAM** 是一款专门用于五轴激光切割加工设计的 CAD/CAM 集成软件系统。该软件结合了计算机辅助设计（CAD）和计算机辅助制造（CAM）功能，提供从 3D 模型导入、路径规划、姿态调整到 NC 程序输出的完整解决方案。

### 核心技术基础

**MyCAM** 基于以下核心技术构建：

| 技术组件 | 版本/说明 | 用途 |
|---------|----------|------|
| **.NET Framework** | 4.8 | 软件运行平台 |
| **C# 语言** | 7.3 | 主要开发语言 |
| **OpenCascade** | 3D 几何内核 | 3D 建模与几何处理 |
| **VNC 协议** | TightVNC 兼容 | 远程连接与监控 |

软件采用 OpenCascade 几何内核，提供强大的 3D 几何处理能力，使用户能够精确地进行加工路径设计和姿态调整。

## 1.2 软件用途与应用场景

### 主要功能模块

**MyCAM** 软件主要提供以下核心功能：

**1. CAD 功能**
- 3D 模型导入与处理（支持 STEP、IGES、STL 格式）
- 模型缝补与检查
- 坐标系转换与调整
- 特征点与特征线创建

**2. CAM 功能**
- 加工路径生成（支持线段、圆弧等多种几何类型）
- 姿态调整与优化
- 路径排序与管理
- 引线、过切、微连设置
- NC 程序输出

**3. 仿真功能**
- 实时加工仿真
- 碰撞检测
- 加工过程可视化

**4. 远程监控**
- VNC 远程连接
- 实时数据传输

### 应用场景

**MyCAM** 适用于以下加工领域：

| 应用类型 | 说明 | 典型工件 |
|---------|------|---------|
| **异形盖件** | 五轴联动激光切割 | 汽车覆盖件、飞机蒙皮 |
| **管拉件** | 管材自动化路径生成 | 排气管、结构管材 |
| **复杂曲面** | 精密空间曲面加工 | 模具、雕刻件 |
| **批量生产** | 多工件自动化排产 | 标准件批量加工 |

## 1.3 系统需求

### 1.3.1 硬件需求

| 硬件 | 最低需求 | 推荐配置 |
|------|----------|----------|
| **处理器** | Intel Core i5 或同级别 | Intel Core i7 或更高 |
| **内存** | 8 GB RAM | 16 GB RAM 或更高 |
| **显卡** | 支持 OpenGL 3.3 以上 | 专业显卡，支持 OpenGL 4.5 |
| **硬盘空间** | 5 GB 可用空间 | 10 GB 以上（含项目文件空间）|
| **屏幕分辨率** | 1920 × 1080 | 2560 × 1440 或更高 |
| **网络** | 可选（用于 VNC 连接）| 100 Mbps 以上有线网络 |

### 1.3.2 软件需求

| 项目 | 需求 |
|------|------|
| **操作系统** | Windows 11（64 位版）|
| **.NET Framework** | .NET Framework 4.8 或更高版本 |
| **OpenCascade** | 已集成到软件（软件内置）|
| **VNC Viewer** | TightVNC 或兼容软件（用于远程连接功能）|

### 1.3.3 其他需求

**授权认证：**
- 软件启动时需通过授权验证
- 如出现授权错误，请联系系统管理员

**VNC 服务器：**
- 需与加工机床在同一网络（用于远程监控功能）
- 需要机床端安装并配置 VNC 服务

**机床配置文件：**
- `MachineData.mac` 文件需正确设置
- 文件位置：软件安装目录下的机床类型文件夹

## 1.4 授权与启动说明

### 1.4.1 授权机制

MyCAM 采用授权验证机制，软件启动时会自动进行授权检查。

**授权检查流程：**

```
软件启动
    ↓
授权验证
    ↓
验证授权状态
    ↓
[授权有效] → 软件正常启动
    ↓
[授权无效] → 显示错误信息并自动关闭
```

> **⚠️ 注意事项：**
> - 如出现授权错误，请联系系统管理员或软件供应商
> - 确保网络连接正常（如使用网络授权）

### 1.4.2 软件启动

**步骤 1：检查前置条件**
- 确认 .NET Framework 4.8 已安装
- 确认显卡驱动程序已更新

**步骤 2：启动软件**
- 双击执行 `MyCAM.exe`
- 系统将自动进行授权验证

**步骤 3：初始化检查**

软件启动后会进行以下初始化：

**1. Viewer 初始化**
- 如初始化失败，会显示信息："初始化 Viewer 错误，请检查显卡是否已安装驱动"
- 请确认 OpenCascade 库已正确安装

**2. 机床数据载入**
- 系统会自动读取 `MachineData.mac` 机床配置文件
- 如读取失败，系统会使用默认机床参数，并显示警告信息

**3. 机床模型载入**
- 根据机床类型（Table / Mix / Spindle）载入对应的 3D 模型文件
- 模型文件位置：软件安装目录下的机床类型文件夹

### 1.4.3 常见启动问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 授权验证失败 | 授权过期或无效 | 联系软件供应商更新授权 |
| Viewer 初始化错误 | 显卡驱动未安装 | 更新显卡驱动程序 |
| 机床数据载入失败 | 配置文件丢失 | 检查 MachineData.mac 文件 |
| .NET 错误 | Framework 版本过低 | 安装 .NET Framework 4.8 |
