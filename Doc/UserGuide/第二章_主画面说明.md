# 第二章 主画面说明

## 本章概要

本章详细介绍 MyCAM 软件的主画面布局和各功能区块，帮助用户熟悉操作环境。

---

## 2.1 画面配置总览

MyCAM 主画面采用模块化设计，由多个功能区块组成，为用户提供直观高效的操作环境。

### 主画面布局

主画面从上到下、从左到右分为以下区域：

| 区域位置 | 名称 | 主要功能 |
|---------|------|---------|
| **顶部** | 功能菜单栏 | 提供文件、CAD、CAM、模拟等主菜单 |
| **上方** | 工具栏区域 | 显示当前模式的功能按钮（3 层工具栏）|
| **左侧** | 对象树面板 | 显示零件、路径等对象的树状列表 |
| **中央** | 3D 视图区 | 显示 3D 模型和加工路径的主视图 |
| **右侧** | 图层控制栏 | 10 个图层的快速切换按钮 |
| **左下方** | 视图工具栏 | 标准视图切换和显示选项 |
| **下方** | 日志面板 | 显示操作提示、警告和错误信息 |
| **底部** | CNC 面板 | CNC 机床的操控窗口（可选）|

> **对应文件名称：** `ch02_01_主画面布局.png`  
> **重要性：** ⭐⭐⭐ 必须

## 2.2 主要功能区块介绍

### 2.2.1 功能菜单栏

位于窗口最顶部的主菜单栏，包含以下菜单：

| 菜单名称 | 功能说明 |
|---------|---------|
| **文件** | 3D 模型导入、项目文件读取和保存 |
| **CAD** | 模型缝补、特征元素创建、坐标转换 |
| **CAM** | 路径生成、编辑、后处理 |
| **模拟** | 加工模拟、碰撞检测功能 |
| **设置** | CNC 连接设置等系统配置 |
| **控制器** | CNC 操控介面 |

### 2.2.2 工具栏区域

包含 4 个主要工具栏，通过菜单切换显示。

> **📸 截图建议 2-2：CAM 工具栏 - 显示 CAM 工具栏的所有功能按钮**  
> **对应文件名称：** `ch02_02_CAM工具栏.png`  
> **重要性：** ⭐⭐⭐ 必须

### 2.2.3 对象树面板

位于主画面左侧，以树状结构显示当前项目中的所有对象。

**树状结构层级：**

```
零件（Part）
├── Part1
├── Part2
├── ...

---

路徑 (Path)
├── Path1
├── Path2
└── ...

- 路徑树状列表中的对象顺序，及代表實際加工顺序

```

**主要功能：**
- 查看所有导入的零件和生成的路径
- 选择对象进行编辑（单击选择）
- 显示或隐藏对象（空白鍵操作）

### 2.2.4 3D 视图区

位于主画面中央，占据最大面积，用于显示 3D 模型和加工路径。

**视图交互操作：**

| 操作 | 鼠标操作 | 功能 |
|------|---------|------|
| **旋转视图** | 鼠标右鍵拖动 | 绕视点旋转视图 |
| **平移视图** | 鼠标中键拖动 | 平移视图位置 |
| **缩放视图** | 鼠标滚轮 | 放大或缩小视图 |
| **选择对象** | 鼠标左键点击 | 选择零件或路径 |

**显示内容：**
- 3D 零件模型（灰白色显示）
- 加工路径（彩色线条，不同图层不同颜色）
- 根據不同編輯模式顯示關鍵 CAM 工藝數據

### 2.2.5 图层控制栏

位于 3D 视图区右侧，提供 10 个图层的快速切换。

**图层按钮：**

| 图层编号 | 颜色 | 用途 |
|---------|------|------|
| **1** | 蓝色 | 默认图层 |
| **2** | 橙色 | 自定义图层 |
| **3** | 紫色 | 自定义图层 |
| **4** | 黄色 | 自定义图层 |
| **5** | 绿色 | 自定义图层 |
| **6** | 番茄红 | 自定义图层 |
| **7** | 黄绿色 | 自定义图层 |
| **8** | 褐色 | 自定义图层 |
| **9** | 洋红色 | 自定义图层 |
| **10** | 青色 | 自定义图层 |


> **💡 提示：**  
> 图层功能可用于区分不同类型的加工路径，详细说明请参阅 [第五章 5.6 图层管理]

### 2.2.6 视图工具栏

位于 3D 视图区左侧，提供标准视图切换和显示选项。

**标准视图按钮：**

| 按钮 | 视图名称 | 说明 |
|------|---------|------|
| **AXO** | 等轴测视图 | 默认的三维视图 |
| **X+** | X 轴正向视图 | 从 X 轴正方向观看 |
| **X-** | X 轴负向视图 | 从 X 轴负方向观看 |
| **Y+** | Y 轴正向视图 | 从 Y 轴正方向观看 |
| **Y-** | Y 轴负向视图 | 从 Y 轴负方向观看 |
| **Z+** | Z 轴正向视图（俯视图）| 从 Z 轴正方向观看 |
| **Z-** | Z 轴负向视图（仰视图）| 从 Z 轴负方向观看 |

**显示选项按钮（可切换）：**

| 功能 | 说明 |
|------|------|
| **显示姿态** | 显示或隐藏刀具姿态向量 |
| **显示顺序** | 显示或隐藏路径顺序编号 |
| **显示路径方向** | 显示或隐藏路径方向箭头 |
| **显示抬下刀路径** | 显示或隐藏空移路径 |

> **💡 提示：**  
> 点击显示选项按钮，按钮会呈现选中状态（高亮），再次点击可取消选中

### 2.2.7 日志面板

位于主画面下方，显示软件运行过程中的各类信息。

**信息类型：**

| 类型 | 颜色标识 | 说明 | 示例 |
|------|---------|------|------|
| **提示信息** | 黑色文字 | 正常操作提示 | "路径已添加" |
| **警告信息** | 橙色文字 | 需要注意的问题 | "重合的第二个点" |
| **错误信息** | 红色文字 | 操作失败或异常 | "导入失败" |

**日志面板功能：**
- 自动显示最新信息（自动滚动到底部）
- 保留历史操作记录
- 支持鼠标滚轮查看历史信息
- 可通过滚动条查看完整日志

### 2.2.8 VNC 远程面板

位于主画面底部，用于远程监控机床的实时状态。

**主要功能：**
- 显示机床端的实时画面
- 远程监控加工进度
- 查看机床运行状态

**使用说明：**
- 需要先配置 VNC 连接设置（参见 [第六章 6.2.1](第六章_其他功能.md#621-vnc-连接设置)）
- 连接成功后，此面板会显示机床端画面
- 可调整面板高度以获得更好的视野

## 1.4 授權與啟動說明

### 1.4.1 授權機制

MyCAM 采用授權驗證機制，軟件啟動時會自動進行授權檢查。

**授權檢查流程：**

```
軟件啟動
    ↓
授權驗證
    ↓
驗證授權狀態
    ↓
[授權有效] → 軟件正常啟動
    ↓
[授權無效] → 顯示錯誤信息並自動關閉
```

> **⚠️ 注意事項：**
> - 如出現授權錯誤，請聯系系統管理員或軟件供應商
> - 確保網絡連接正常（如使用網絡授權）

### 1.4.2 軟件啟動

**步驟 1：檢查前置條件**
- 確認 .NET Framework 4.8 已安裝
- 確認顯卡驅動程序已更新

**步驟 2：啟動軟件**
- 雙擊執行 `MyCAM.exe`
- 系統將自動進行授權驗證

**步驟 3：初始化檢查**

軟件啟動後會進行以下初始化：

**1. Viewer 初始化**
- 如初始化失敗，會顯示信息："初始化 Viewer 錯誤，請檢查顯卡是否已安裝驅動"
- 請確認 OpenCascade 庫已正確安裝

**2. 機床數據載入**
- 系統會自動讀取 `MachineData.mac` 機床配置文件
- 如讀取失敗，系統會使用默認機床參數，並顯示警告信息

**3. 機床模型載入**
- 根據機床類型（Table / Mix / Spindle）載入對應的 3D 模型文件
- 模型文件位置：軟件安裝目錄下的機床類型文件夾

> **📸 截圖建議 1-3：啟動界面 - 軟件成功啟動後的初始界面，顯示各功能區已正常載入**  
> **對應文件名稱：** `ch01_03_啟動界面.png`  
> **重要性：** ⭐⭐ 建議

### 1.4.3 常見啟動問題

| 問題 | 可能原因 | 解決方案 |
|------|---------|---------|
| 授權驗證失敗 | 授權過期或無效 | 聯系軟件供應商更新授權 |
| Viewer 初始化錯誤 | 顯卡驅動未安裝 | 更新顯卡驅動程序 |
| 機床數據載入失敗 | 配置文件丟失 | 檢查 MachineData.mac 文件 |
| .NET 錯誤 | Framework 版本過低 | 安裝 .NET Framework 4.8 |

**詳細的錯誤處理說明，請參考：**
- 第四章 - 錯誤信息與處理

## 2.3 画面配置调整

MyCAM 支持灵活的画面布局调整，适应不同的屏幕尺寸和使用习惯。

### 2.3.1 面板大小调整

**调整方法：**
- 各面板之间有分隔线，可以拖动调整大小
- 拖动对象树面板右边界，调整树面板宽度
- 拖动日志面板上边界，调整日志面板高度
- 拖动 VNC 面板上边界，调整 VNC 面板高度

**布局特点：**
- 对象树面板默认占窗口宽度的 15%
- 3D 视图区默认占窗口宽度的 85%
- 日志面板默认占窗口高度的 6%
- VNC 面板默认占窗口高度的 47%

> **💡 提示：**  
> 软件会记住用户调整的面板大小，下次启动时自动恢复

### 2.3.2 工具栏显示控制

**工具栏切换：**
- 工具栏会根据当前选择的菜单自动切换
- 第二层和第三层工具栏根据操作动态显示或隐藏
- 不使用的工具栏会自动隐藏，节省画面空间

## 2.4 视图控制

### 2.4.1 标准视图切换

通过视图工具栏的按钮，可快速切换到预定义的标准视图。

**各视图说明：**

**AXO（等轴测视图）：**
- 默认的三维视图
- 可同时看到 X、Y、Z 三个轴向
- 适合整体观察模型和路径

**X+/X-（X 轴视图）：**
- X+ 视图：从 X 轴正方向观看（右视图）
- X- 视图：从 X 轴负方向观看（左视图）
- 适合检查 YZ 平面上的特征

**Y+/Y-（Y 轴视图）：**
- Y+ 视图：从 Y 轴正方向观看（主视图）
- Y- 视图：从 Y 轴负方向观看（后视图）
- 适合检查 XZ 平面上的特征

**Z+/Z-（Z 轴视图）：**
- Z+ 视图：从 Z 轴正方向观看（俯视图）
- Z- 视图：从 Z 轴负方向观看（仰视图）
- 适合检查 XY 平面上的特征

> **对应文件名称：** `ch02_04_视图切换示例.png`  
> **重要性：** ⭐⭐ 建议

### 2.4.2 显示选项控制

通过视图工具栏底部的 4 个切换按钮，控制视图中显示的内容。

**显示姿态：**
- 开启：在每个路径点显示刀具姿态向量（红色箭头）
- 关闭：隐藏姿态向量，视图更清爽
- 用途：检查刀具方向是否正确

**显示顺序：**
- 开启：在每条路径旁显示加工顺序编号
- 关闭：隐藏顺序编号
- 用途：确认加工顺序是否符合要求

**显示路径方向：**
- 开启：在路径上显示方向箭头（蓝色）
- 关闭：隐藏方向箭头
- 用途：检查路径切割方向

**显示抬下刀路径：**
- 开启：显示空移路径（虚线）
- 关闭：只显示切割路径
- 用途：查看完整的刀具运动轨迹

## 2.5 画面操作技巧

### 2.5.1 鼠标操作总结

| 功能 | 操作方法 | 说明 |
|------|---------|------|
| 旋转视图 | 鼠标中键拖动 | 三维旋转观察 |
| 平移视图 | Ctrl + 鼠标中键拖动 | 上下左右移动视图 |
| 缩放视图 | 鼠标滚轮 | 放大或缩小 |
| 框选缩放 | Ctrl + 鼠标右键拖动 | 精确放大特定区域 |
| 选择对象 | 鼠标左键点击 | 选择零件或路径 |

### 2.5.2 快捷键

MyCAM 提供多种快捷键提高操作效率：

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 保存项目 | Ctrl + S | 快速保存当前项目 |
| 取消操作 | Esc | 退出当前操作模式 |

> **💡 提示：**  
> 完整的快捷键列表请参阅 [附录 A - 快捷键列表](附录A_快捷键列表.md)

### 2.5.3 画面自定义

**可调整项目：**
- 各面板的大小比例
- 视图显示选项的默认状态
- 对象树面板的宽度

**设置保存：**
- 软件会自动保存用户的画面设置
- 下次启动时自动恢复上次的画面布局
