# 第二章 主画面说明

## 本章概要

本章详细介绍 MyCAM 软件的主画面布局和各功能区块，帮助用户熟悉操作环境。

---

## 2.1 画面配置总览

MyCAM 主画面采用模块化设计，由多个功能区块组成，为用户提供直观高效的操作环境。

### 主画面布局

主画面从上到下、从左到右分为以下区域：

| 区域位置 | 名称 | 主要功能 |
|---------|------|---------|
| **顶部** | 功能菜单栏 | 提供文件、CAD、CAM、模拟等主菜单 |
| **上方** | 工具栏区域 | 显示当前模式的功能按钮（3 层工具栏）|
| **左侧** | 对象树面板 | 显示零件、路径等对象的树状列表 |
| **中央** | 3D 视图区 | 显示 3D 模型和加工路径的主视图 |
| **右侧** | 图层控制栏 | 10 个图层的快速切换按钮 |
| **左下方** | 视图工具栏 | 标准视图切换和显示选项 |
| **下方** | 日志面板 | 显示操作提示、警告和错误信息 |
| **底部** | CNC 面板 | CNC 机床的操控窗口（可选）|

## 2.2 主要功能区块介绍

### 2.2.1 功能菜单栏

位于窗口最顶部的主菜单栏，包含以下菜单：

| 菜单名称 | 功能说明 |
|---------|---------|
| **文件** | 3D 模型导入、项目文件读取和保存 |
| **CAD** | 模型缝补、特征元素创建、坐标转换 |
| **CAM** | 路径生成、编辑、后处理 |
| **模拟** | 加工模拟、碰撞检测功能 |
| **设置** | CNC 连接设置等系统配置 |
| **控制器** | CNC 操控介面 |

### 2.2.2 工具栏区域

包含 4 个主要工具栏，通过菜单切换显示。

> **📸 截图建议 2-2：CAM 工具栏 - 显示 CAM 工具栏的所有功能按钮**  
> **对应文件名称：** `ch02_02_CAM工具栏.png`  
> **重要性：** ⭐⭐⭐ 必须

### 2.2.3 对象树面板

位于主画面左侧，以树状结构显示当前项目中的所有对象。

**树状结构层级：**

```
零件（Part）
├── Part1
├── Part2
├── ...

---

路徑 (Path)
├── Path1
├── Path2
└── ...

- 路徑树状列表中的对象顺序，及代表實際加工顺序

```

**主要功能：**
- 查看所有导入的零件和生成的路径
- 选择对象进行编辑（单击选择）
- 显示或隐藏对象（空白鍵操作）

### 2.2.4 3D 视图区

位于主画面中央，占据最大面积，用于显示 3D 模型和加工路径。

**视图交互操作：**

| 操作 | 鼠标操作 | 功能 |
|------|---------|------|
| **旋转视图** | 鼠标右鍵拖动 | 绕视点旋转视图 |
| **平移视图** | 鼠标中键拖动 | 平移视图位置 |
| **缩放视图** | 鼠标滚轮 | 放大或缩小视图 |
| **选择对象** | 鼠标左键点击 | 选择零件或路径 |

**显示内容：**
- 3D 零件模型（灰白色显示）
- 加工路径（彩色线条，不同图层不同颜色）
- 根據不同編輯模式顯示關鍵 CAM 工藝數據

### 2.2.5 图层控制栏

位于 3D 视图区右侧，提供 10 个图层的快速切换。

**图层按钮：**

| 图层编号 | 颜色 | 用途 |
|---------|------|------|
| **1** | 蓝色 | 默认图层 |
| **2** | 橙色 | 自定义图层 |
| **3** | 紫色 | 自定义图层 |
| **4** | 黄色 | 自定义图层 |
| **5** | 绿色 | 自定义图层 |
| **6** | 番茄红 | 自定义图层 |
| **7** | 黄绿色 | 自定义图层 |
| **8** | 褐色 | 自定义图层 |
| **9** | 洋红色 | 自定义图层 |
| **10** | 青色 | 自定义图层 |


> **💡 提示：**  
> 图层功能可用于区分不同类型的加工路径，详细说明请参阅 [第五章 5.6 图层管理]

### 2.2.6 视图工具栏

位于 3D 视图区左侧，提供标准视图切换和显示选项。

**标准视图按钮：**

| 按钮 | 视图名称 | 说明 |
|------|---------|------|
| **AXO** | 等轴测视图 | 默认的三维视图 |
| **X+** | X 轴正向视图 | 从 X 轴正方向观看 |
| **X-** | X 轴负向视图 | 从 X 轴负方向观看 |
| **Y+** | Y 轴正向视图 | 从 Y 轴正方向观看 |
| **Y-** | Y 轴负向视图 | 从 Y 轴负方向观看 |
| **Z+** | Z 轴正向视图（俯视图）| 从 Z 轴正方向观看 |
| **Z-** | Z 轴负向视图（仰视图）| 从 Z 轴负方向观看 |

**显示选项按钮（可切换）：**

| 功能 | 说明 |
|------|------|
| **显示姿态** | 显示或隐藏刀具姿态向量 |
| **显示顺序** | 显示或隐藏路径顺序编号 |
| **显示路径方向** | 显示或隐藏路径方向箭头 |
| **显示抬下刀路径** | 显示或隐藏空移路径 |

> **💡 提示：**  
> 点击显示选项按钮，按钮会呈现选中状态（高亮），再次点击可取消选中

### 2.2.7 日志面板

位于主画面下方，显示软件运行过程中的各类信息。

**信息类型：**

| 类型 | 颜色标识 | 说明 | 示例 |
|------|---------|------|------|
| **提示信息** | 黑色文字 | 正常操作提示 | "路径已添加" |
| **警告信息** | 橙色文字 | 需要注意的问题 | "重合的第二个点" |
| **错误信息** | 红色文字 | 操作失败或异常 | "导入失败" |

**日志面板功能：**
- 自动显示最新信息（自动滚动到底部）
- 支持鼠标滚轮查看历史信息
- 可通过滚动条查看完整日志

> **重要性：** ⭐⭐ 建议

### 2.2.8 CNC 操作面板

位于主画面底部，用于控制 CNC。

**主要功能：**
- 显示机床端的实时画面
- 远程监控加工进度
- 查看机床运行状态

**使用说明：**
- 需要先配置 CNC 连接设置（参见 [第六章 6.2.1]）
- 连接成功后，此面板会显示机床端画面

## 2.4 视图控制

### 2.4.1 标准视图切换

通过视图工具栏的按钮，可快速切换到预定义的标准视图。

**各视图说明：**

**AXO（等轴测视图）：**
- 默认的三维视图
- 可同时看到 X、Y、Z 三个轴向
- 适合整体观察模型和路径

**X+/X-（X 轴视图）：**
- X+ 视图：从 X 轴正方向观看（右视图）
- X- 视图：从 X 轴负方向观看（左视图）
- 适合检查 YZ 平面上的特征

**Y+/Y-（Y 轴视图）：**
- Y+ 视图：从 Y 轴正方向观看（主视图）
- Y- 视图：从 Y 轴负方向观看（后视图）
- 适合检查 XZ 平面上的特征

**Z+/Z-（Z 轴视图）：**
- Z+ 视图：从 Z 轴正方向观看（俯视图）
- Z- 视图：从 Z 轴负方向观看（仰视图）
- 适合检查 XY 平面上的特征

### 2.4.2 显示选项控制

通过视图工具栏底部的 4 个切换按钮，控制视图中显示的内容。

**显示姿态：**
- 开启：在每个路径点显示刀具姿态向量
- 关闭：隐藏姿态向量，视图更清爽
- 用途：检查刀具方向是否正确

**显示顺序：**
- 开启：在每条路径旁显示加工顺序编号
- 关闭：隐藏顺序编号
- 用途：确认加工顺序是否符合要求

**显示路径方向：**
- 开启：在路径上显示方向箭头
- 关闭：隐藏方向箭头
- 用途：检查路径切割方向

**显示抬下刀路径：**
- 开启：显示空移路径
- 关闭：只显示切割路径
- 用途：查看完整的刀具运动轨迹
