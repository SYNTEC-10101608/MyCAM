# 第二章 主畫面說明

## 2.1 介面配置總覽

MyCAM 軟體主畫面採用分區式設計，將不同功能區域清楚劃分，讓使用者能快速找到所需功能。主畫面由上至下、由左至右分為八個主要區域：

| 區域名稱 | 位置 | 主要功能 |
|---------|------|----------|
| 功能選單列 | 最上方 | 切換不同功能模式（檔案/CAD/CAM/模擬/設定/控制器） |
| 工具列區域 | 選單列下方 | 提供目前模式下的所有功能按鈕 |
| 物件樹狀檢視區 | 左側 | 顯示專案中的零件與加工路徑清單 |
| 3D 視圖區 | 中央區域 | 顯示 3D 模型與加工路徑的視覺化畫面 |
| 圖層控制列 | 3D 視圖區右側 | 管理加工路徑的圖層分類（CAM 模式專用） |
| 視圖工具列 | 3D 視圖區左側 | 快速切換視角與顯示選項 |
| 日誌訊息區 | 下方橫條區域 | 顯示系統訊息、警告與錯誤資訊 |
| VNC 遠端連線區 | 底部（直式螢幕） | 顯示遠端 CNC 控制器畫面 |

> **📷 截圖建議 2-1：** 請在此處放置 MyCAM 軟體主畫面完整截圖，並使用紅框或箭頭標示上述 8 個區域的位置與名稱。

## 2.2 主要功能區塊介紹

### 2.2.1 功能選單列

功能選單列位於軟體最上方，用於切換不同的工作模式。選單列包含以下項目：

**主要選單項目：**

- **檔案**：提供 3D 模型匯入、專案檔匯入與儲存等檔案管理功能
- **CAD**：進入 CAD 模式，提供模型編輯、座標轉換等幾何處理功能
- **CAM**：進入 CAM 模式，提供路徑規劃、刀具姿態調整、NC 程式轉出等加工相關功能
- **模擬**：進入模擬模式，提供加工過程模擬與碰撞檢測功能
- **設定**：包含 VNC 連線設定等系統設定選項
- **控制器**：開啟遠端控制器操作視窗（橫式螢幕模式）

**功能切換機制：**

當您點選不同的選單項目時，下方的工具列區域會自動切換，顯示對應模式的功能按鈕。例如：
- 點選「CAD」→ 下方顯示 CAD 功能工具列（模型縫補、新增特徵點、座標轉換等）
- 點選「CAM」→ 下方顯示 CAM 功能工具列（新增路徑、起點、引線、過切等）
- 點選「模擬」→ 下方顯示模擬控制工具列（播放、暫停、停止等）

> **📷 截圖建議 2-2：** 請放置選單列特寫，清楚顯示所有選單項目。

### 2.2.2 工具列區域

工具列區域位於功能選單列下方，會根據目前選擇的功能模式動態顯示不同的工具列。

#### 工具列切換機制

**檔案模式：**
- 提供 3D 模型匯入、專案檔匯入與儲存等檔案管理功能

**CAD 模式：**
- 提供模型處理、特徵點建立、座標轉換等幾何編輯功能

**CAM 模式：**
- 提供路徑管理、路徑編輯、刀具姿態調整、加工參數設定、後處理等功能

**模擬模式：**
- 提供播放控制、路徑導航、碰撞檢測、速度調整等模擬功能

> **💡 提示：** 各模式的詳細功能操作說明，請參考第四章（CAD 功能）、第五章（CAM 功能）與第六章（其他功能）。

### 2.2.3 物件樹狀檢視區

物件樹狀檢視區位於軟體左側，以樹狀結構顯示專案中的所有物件，方便使用者快速選取與管理。

#### 樹狀結構

樹狀檢視區包含兩個主要節點：

**Part（零件）節點：**
- 顯示所有匯入的 3D 模型零件
- 每個零件以獨立節點呈現
- 可展開查看零件細節（如有子元件）

**Path（路徑）節點：**
- 顯示所有建立的加工路徑
- 路徑依照順序編號顯示
- 展開節點可查看路徑類型與圖層資訊

#### 選取與互動方式

**選取方式：**
- **單擊**：選取單一物件
- **Ctrl + 單擊**：多選物件（可選取多個物件）
- **選取後效果**：
  - 3D 視圖區會高亮顯示選取的物件
  - 工具列的功能按鈕會根據選取內容啟用或停用
  - 可對選取的物件執行編輯操作

> **📷 截圖建議 2-5：** 請放置樹狀檢視區截圖，展示 Part 和 Path 兩個主節點以及其下的子節點結構。

### 2.2.4 3D 視圖區

3D 視圖區是軟體的主要工作區域，位於畫面中央，用於顯示 3D 模型、加工路徑以及模擬動畫。視圖區採用 OpenCascade 3D 渲染引擎，提供高品質的 3D 視覺化效果。

#### 顯示內容

**根據不同模式顯示：**
- **檔案/CAD 模式**：顯示 3D 零件模型、特徵點、特徵線
- **CAM 模式**：顯示零件模型、加工路徑、刀具姿態、路徑順序、空移路徑
- **模擬模式**：顯示機構動畫、加工過程、碰撞檢測結果

#### 滑鼠操作方式

| 操作 | 功能 |
|------|------|
| **左鍵拖曳** | 旋轉視角，觀察模型不同角度 |
| **中鍵拖曳** | 平移視圖，移動觀察位置 |
| **滾輪向上** | 放大視圖，靠近模型 |
| **滾輪向下** | 縮小視圖，遠離模型 |
| **左鍵單擊** | 選取物件（在特定功能下） |
| **左鍵雙擊** | 快速選取連續物件（在特定功能下） |

#### 座標系統顯示

3D 視圖區左下角會顯示 G54 座標系統的三軸方向：
- **X 軸**：紅色箭頭
- **Y 軸**：綠色箭頭
- **Z 軸**：藍色箭頭

此座標系統可幫助您了解目前的觀察方向與模型擺放方位。

> **📷 截圖建議 2-6：** 請放置 3D 視圖區截圖，顯示零件模型與加工路徑，並標示座標系統位置。

### 2.2.5 圖層控制列

圖層控制列位於 3D 視圖區的右側，是一個垂直排列的按鈕列。此功能僅在 **CAM 模式**下顯示，用於管理加工路徑的圖層分類。

#### 圖層功能說明

軟體提供 10 個圖層，讓您可以將不同類型或階段的加工路徑分類管理。每個圖層都有專屬的顏色，方便在 3D 視圖區中識別。

#### 圖層顏色對照

| 圖層編號 | 按鈕顯示 | 顏色 |
|---------|---------|------|
| 圖層 1 | 1 | 藍色 |
| 圖層 2 | 2 | 橙色 |
| 圖層 3 | 3 | 紫色 |
| 圖層 4 | 4 | 黃色 |
| 圖層 5 | 5 | 綠色 |
| 圖層 6 | 6 | 番茄紅 |
| 圖層 7 | 7 | 黃綠色 |
| 圖層 8 | 8 | 褐色 |
| 圖層 9 | 9 | 洋紅色 |
| 圖層 10 | 10 | 青色 |

#### 使用方式

1. 在樹狀檢視區選取要設定圖層的路徑
2. 點選圖層控制列上的圖層按鈕（1-10）
3. 選取的路徑會被指派到該圖層，並在 3D 視圖區以對應顏色顯示

> **💡 提示：** 合理使用圖層可以幫助您組織複雜的加工專案，例如將不同加工階段或不同刀具的路徑分配到不同圖層。

> **📷 截圖建議 2-7：** 請放置圖層控制列截圖，清楚顯示 10 個圖層按鈕及其顏色。

### 2.2.6 視圖工具列

視圖工具列位於 3D 視圖區的左側，是一個垂直排列的工具列，提供快速切換視角與顯示選項的功能。

#### 視圖方向切換按鈕

工具列上方提供 7 個標準視角按鈕，讓您快速切換到常用的觀察角度：

| 按鈕 | 視角名稱 | 說明 |
|------|---------|------|
| **AXO** | 等角視圖 | 立體等角視角，便於觀察整體結構 |
| **X+** | 右視圖 | 從 X 軸正向觀察 |
| **X-** | 左視圖 | 從 X 軸負向觀察 |
| **Y+** | 主視圖 | 從 Y 軸正向觀察（正面） |
| **Y-** | 後視圖 | 從 Y 軸負向觀察（背面） |
| **Z+** | 俯視圖 | 從 Z 軸正向觀察（由上往下） |
| **Z-** | 仰視圖 | 從 Z 軸負向觀察（由下往上） |

點選任一視角按鈕後，視圖會自動旋轉到對應方向並縮放至適當大小。

#### 顯示選項開關（CAM 模式專用）

工具列下方提供 4 個顯示選項開關，僅在 **CAM 模式**下可見。這些按鈕採用「按下/彈起」的切換式設計，預設為開啟狀態：

| 按鈕圖示 | 功能名稱 | 說明 | 預設狀態 |
|---------|---------|------|----------|
| 姿態圖示 | **顯示姿態** | 顯示/隱藏刀具姿態向量 | 開啟 |
| 順序圖示 | **顯示順序** | 顯示/隱藏路徑順序編號 | 開啟 |
| 方向圖示 | **顯示路徑方向** | 顯示/隱藏路徑方向箭頭 |  開啟 |
| 空移圖示 | **顯示抬下刀路徑** | 顯示/隱藏空移路徑（虛線顯示） |  開啟 |

**使用方式：**
- 點選按鈕：切換開啟/關閉狀態
- 按鈕按下（深色）：功能開啟
- 按鈕彈起（淺色）：功能關閉
- 即時更新：切換後 3D 視圖區會立即反映變化

> **📷 截圖建議 2-8：** 請放置視圖工具列完整截圖，標示視角切換按鈕與顯示選項開關的位置。

### 2.2.7 日誌訊息區

日誌訊息區位於主畫面下方，是一個橫向的訊息顯示區域。此區域會即時顯示軟體運作過程中的各種訊息，包括操作提示、警告與錯誤資訊。

#### 訊息類型

日誌訊息分為三種類型，透過不同的顏色或圖示區分：

| 訊息類型 | 圖示 | 說明 | 範例 |
|---------|------|------|------|
| **提示訊息** | ?? | 一般操作提示與系統資訊 | 「[操作提醒]請先選擇路徑」 |
| **警告訊息** | ?? | 需要注意的警告訊息 | 「使用默認機構專案檔」 |
| **錯誤訊息** | ? | 嚴重錯誤，需要處理 | 「初始化 Viewer 錯誤，請檢查套件是否安裝齊全」 |

#### 訊息顯示特性

- **自動捲動**：新訊息出現時會自動捲動到最下方
- **訊息保留**：所有訊息會保留在面板中，可向上捲動查看歷史訊息
- **時間戳記**：部分訊息會包含時間資訊
- **操作提醒**：當執行功能時缺少必要條件，會顯示提示訊息

#### 常見訊息範例

**操作提示：**
```
[操作提醒]請先選擇路徑
[操作提醒]只能選擇一個路徑進行編輯
```

**警告訊息：**
```
使用默認機構專案檔
機構樹結構錯誤,使用默認機構樹
```

**錯誤訊息：**
```
初始化 Viewer 錯誤，請檢查套件是否安裝齊全
讀取機構檔案失敗：[錯誤詳情]
```

> **⚠️ 重要：** 當出現錯誤訊息時，請參考第四章「錯誤訊息與處理」以了解詳細的解決方法。

### 2.2.8 VNC 遠端連線區

VNC 遠端連線區用於顯示遠端 CNC 控制器的操作畫面，讓您可以在 MyCAM 軟體中直接監控與操作 CNC 設備。

#### 顯示條件

此區域**僅在特定條件下顯示**：

1. **螢幕方向為直式**：當視窗高度大於寬度時
2. **已建立 VNC 連線**：透過「設定」→「VNC 連線設定」成功連接到 CNC 控制器

#### 功能說明

**在直式螢幕模式下：**
- VNC 遠端連線區會顯示在畫面底部
- 可直接在此區域操作 CNC 控制器介面
- 支援滑鼠點擊、鍵盤輸入等互動操作

**在橫式螢幕模式下：**
- VNC 遠端連線區會自動隱藏
- 改由「控制器」選單項目開啟獨立的控制器操作視窗
- 獨立視窗可自由移動與調整大小

#### 連線設定

**建立 VNC 連線步驟：**
1. 點選選單列的「設定」→「VNC 連線設定」
2. 在對話框中輸入 CNC 控制器的 IP 位址
3. 點選「確認」建立連線
4. 連線成功後，VNC 畫面會顯示在此區域（直式模式）或獨立視窗（橫式模式）

> **⚠️ 注意事項：**
> - VNC 執行檔需與 MyCAM.exe 放置於同一資料夾
> - 建議使用 TightVNC 或相容軟體
> - 確保 CNC 控制器與電腦在同一網路環境下
> - 網路速度建議 100 Mbps 以上以確保操作流暢

## 2.3 畫面配置調整

MyCAM 軟體能夠根據螢幕方向自動調整畫面配置，以適應不同的使用環境與需求。

### 直式螢幕模式（高度 > 寬度）

當視窗高度大於寬度時，軟體會自動切換為直式螢幕配置。

**配置特性：**
- **顯示 VNC 遠端連線區**：畫面底部會顯示 VNC 連線區域
- **三區域佈局**：畫面分為三個主要區域
  - CAD/CAM 工作區
  - 日誌訊息區
  - VNC 遠端連線區

**適用情境：**
- 使用直立式螢幕工作
- 需要同時查看加工路徑與控制器畫面
- 操作員站立操作時的便利配置
