## 2.1 主畫面配置

MyCAM 主畫面採用分區設計，主要包含以下五個區域：

| 區域 | 位置 | 主要功能 |
|------|------|---------|
| **選單列** | 視窗頂部 | 檔案、CAD、CAM、模擬等功能選單 |
| **工具列** | 選單列下方 | 常用功能快速按鈕 |
| **物件樹狀檢視區** | 視窗左側 | 顯示零件（Part）與路徑（Path）樹狀結構 |
| **3D 視圖區** | 視窗中央 | 3D 模型與加工路徑顯示區域 |
| **日誌訊息區** | 視窗底部 | 顯示系統訊息、警告、錯誤等資訊 |

## 2.2 選單列

選單列位於視窗頂部，提供所有主要功能的入口。

### 主要選單項目

**1. 檔案（File）**
- 匯入 3D 模型（STEP / IGES）
- 開啟專案檔（.sfa）
- 儲存專案檔
- 轉出 NC 程式

**2. CAD**
- 模型縫補
- 特徵點建立（圓弧中心、線段中心、兩點中心、包圍盒中心）
- 特徵線建立（兩點連線）
- 工件對齊（面重合、面平行、軸重合、軸平行、點重合）
- 三點轉換
- 軸向轉換

**3. CAM**
- 新增路徑（自動選面、手動選面）
- 起點設定
- 刀具姿態調適
- 過切設定
- 引線設定
- 微連設定
- 空移參數
- 進入/退出點
- 路徑順序
- 圖層管理

**4. 模擬（Simulation）**
- 開始模擬
- 播放控制
- 碰撞檢查

**5. 設定（Settings）**
- 機台設定
- CNC IP 設定
- 其他系統設定

### 工具列

工具列位於選單列下方，提供常用功能的快速存取按鈕。

**工具列按鈕分類：**
- **檔案操作：** 匯入、開啟、儲存、轉出 NC
- **CAD 功能：** 縫補、特徵建立、對齊
- **CAM 功能：** 新增路徑、起點、姿態、參數設定
- **模擬功能：** 開始模擬、播放控制

## 2.3 物件樹狀檢視區

物件樹狀檢視區位於視窗左側，以樹狀結構顯示專案中的所有物件。

### 樹狀結構

```
Part（零件節點）
├── Part_1（零件 1）
├── Part_2（零件 2）
├── Feature_Point_1（特徵點）
└── Feature_Line_1（特徵線）

Path（路徑節點）
├── Path_1（路徑 1）
├── Path_2（路徑 2）
└── Path_3（路徑 3）
```

### Part 節點

**Part 節點**包含所有零件與特徵物件：

- **零件物件**：匯入的 3D 模型，命名為 Part_1, Part_2...
- **特徵點**：建立的輔助參考點，命名為 Feature_Point_1, Feature_Point_2...
- **特徵線**：建立的輔助參考線，命名為 Feature_Line_1, Feature_Line_2...

**操作方式：**
- 點選節點可選取物件
- 雙擊節點可展開/摺疊
- 按住 `Ctrl` 可多選
- 按 `Delete` 可刪除選取的物件

### Path 節點

**Path 節點**包含所有加工路徑：

- **路徑物件**：建立的加工路徑，命名為 Path_1, Path_2...
- **路徑順序**：路徑編號代表加工順序

**操作方式：**
- 點選路徑可選取並高亮顯示
- 可多選路徑進行批次操作
- 按 `Delete` 可刪除選取的路徑

## 2.4 3D 視圖區

3D 視圖區位於視窗中央，是最主要的工作區域，用於顯示 3D 模型與加工路徑。

### 顯示內容

**視圖區顯示的物件：**

| 物件類型 | 顯示顏色 | 說明 |
|---------|---------|------|
| **零件模型** | 黃色 | 匯入的 3D 模型 |
| **特徵點** | 黃色小點 | CAD 特徵點 |
| **特徵線** | 黃色線條 | CAD 特徵線 |
| **加工路徑** | 彩色線條 | CAM 加工路徑（依圖層顯示不同顏色） |
| **刀具向量** | 紅色箭頭 | 刀具姿態方向 |
| **機台模型** | 灰色 | 模擬模式下的機台 3D 模型 |

### 視角控制

**滑鼠操作：**

| 操作 | 滑鼠動作 | 功能 |
|------|---------|------|
| **旋轉視角** | 中鍵拖曳 | 環繞觀察模型 |
| **平移視圖** | Shift + 中鍵拖曳 | 移動視點 |
| **縮放視圖** | 滾輪 | 拉近/拉遠 |
| **框選放大** | 右鍵拖曳 | 框選區域放大 |
| **選取物件** | 左鍵點選 | 選取零件或路徑 |

**快捷鍵：**

| 快捷鍵 | 功能 |
|--------|------|
| `F` | 適應視窗（Fit） |
| `Home` | 復位視角 |
| `Esc` | 取消目前操作 |
| `Delete` | 刪除選取物件 |

### 視圖模式

**標準視角：**
- 前視圖（Front）
- 後視圖（Back）
- 左視圖（Left）
- 右視圖（Right）
- 頂視圖（Top）
- 底視圖（Bottom）
- 等角視圖（Isometric）

## 2.5 日誌訊息區

日誌訊息區位於視窗底部，用於顯示系統操作訊息、警告、錯誤等資訊。

### 訊息類型

**日誌訊息分為以下類型：**

| 訊息類型 | 顯示顏色 | 說明 | 範例 |
|---------|---------|------|------|
| **提示訊息** | 黑色 | 一般操作提示 | 「匯入成功」 |
| **警告訊息** | 橙色 | 需要注意的情況 | 「模型存在破面」 |
| **錯誤訊息** | 紅色 | 操作失敗或錯誤 | 「匯入失敗」 |
| **操作提醒** | 藍色 | 操作指引 | 「[操作提醒] 請先選擇零件」 |

### 常見訊息

**成功訊息：**
- 「匯入成功」
- 「專案檔儲存成功」
- 「NC 檔案轉出成功」
- 「模擬計算完成」

**警告訊息：**
- 「模型存在破面，建議執行縫補」
- 「路徑數量過多，可能影響效能」
- 「刀具姿態角度過大」

**錯誤訊息：**
- 「匯入失敗」
- 「尚無路徑能夠參與模擬」
- 「授權驗證失敗」
- 「初始化 Viewer 錯誤」

**操作提醒：**
- 「[操作提醒] 請先選擇零件」
- 「[操作提醒] 請先建立加工路徑」
- 「[操作提醒] 請先設定刀具姿態」

### 使用技巧

**日誌訊息區操作：**
- 訊息會自動捲動顯示最新內容
- 可手動捲動查看歷史訊息
- 重要訊息會以顏色區分
- 建議定期檢查日誌，確認操作狀態

## 2.6 工作流程說明

### 標準工作流程

**一般加工專案的操作流程：**

1. **CAD 階段**
   - 匯入 3D 模型（檔案選單）
   - 檢查模型完整性（日誌訊息區）
   - 執行模型縫補（如需要）
   - 建立特徵點/線（如需要）
   - 工件對齊（調整零件位置與姿態）

2. **CAM 階段**
   - 新增加工路徑（自動或手動選面）
   - 設定起點與方向
   - 調整刀具姿態
   - 設定過切、引線、微連
   - 設定空移參數
   - 調整路徑順序

3. **模擬階段**
   - 開始模擬
   - 檢查碰撞
   - 調整參數（如有碰撞）
   - 重新模擬驗證

4. **輸出階段**
   - 轉出 NC 程式
   - 傳輸至 CNC 控制器（如有遠端連線）

### 界面切換

**模式切換：**
- 點選選單列的「CAD」進入 CAD 模式
- 點選選單列的「CAM」進入 CAM 模式
- 點選選單列的「模擬」進入模擬模式

**模式特性：**
- 不同模式下，工具列會顯示對應的功能按鈕
- 物件樹會顯示對應的節點
- 日誌訊息會提示目前模式
