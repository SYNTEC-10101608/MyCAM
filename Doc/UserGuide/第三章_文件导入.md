# 第三章 文件导入

## 本章概要

本章介绍 MyCAM 软件的文件导入与项目管理功能，帮助用户快速开始加工任务。

---

## 3.1 3D 模型导入

### 3.1.1 支持的文件格式

MyCAM 支持导入以下 3D 模型格式：

| 格式 | 扩展名 | 精度 | 推荐 | 适用场景 |
|------|-------|------|--------|---------|
| **STEP** | .step, .stp | 高 | 是 | 精密加工，保留完整几何信息 |
| **IGES** | .iges, .igs | 低 | 否 | 較輕量的 3D 圖檔格式，信息可能丟失 |

> **💡 格式选择建议：**  
> 优先使用 STEP 格式，可保留完整的拓扑结构和几何精度。IGES 格式兼容性好但精度略低。

> **📸 截图建议 3-1：3D 模型导入 - 显示导入对话框和成功导入后的模型显示**  
> **对应文件名称：** `ch03_01_3D模型导入.png`  
> **重要性：** ⭐⭐⭐ 必须

### 3.1.2 导入操作步骤

**步骤流程：**

1. 点击顶部"文件"菜单
2. 点击工具栏"3D模型"按钮
3. 在文件对话框中选择模型文件
4. 点击"打开"按钮或雙擊模型文件

**导入成功标志：**
- ✅ 3D 模型显示在中央视图区（灰白色）
- ✅ 对象树"零件"节点下添加新模型
- ✅ 日志面板显示导入成功信息

> **⚠️ 注意事项：**
> - 大型模型导入可能需要数十秒至數分鐘
> - 导入过程中请勿关闭软件
> - 建议在源 CAD 編輯軟件中预先检查模型质量

### 3.1.3 常见导入问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 显示"导入失败" | 文件损坏或格式错误 | 重新导出模型，确认文件扩展名 |
| 模型显示不完整 | 包含无效几何 | 在源 CAD 中修复模型 |
| 加载时间过长 | 模型过于复杂 | 简化模型或分割导入 |
| 模型位置偏移 | 坐标系不匹配 | 使用坐标转换功能调整 |

## 3.2 项目文件管理

MyCAM 使用 .sfa 格式保存完整的项目数据，方便后续编辑和共享。

### 3.2.1 项目文件说明

**项目文件（.sfa）包含的数据：**

| 数据类型 | 是否保存 | 说明 |
|---------|---------|------|
| 3D 模型几何 | ✅ 是 | 完整的零件几何数据 |
| 加工路径 | ✅ 是 | 所有生成的路径信息 |
| 加工工藝 | ✅ 是 | 每个路径点的姿态向量 |
| 三点校正数据 | ✅ 是 | 坐标转换参数 |
| 视图状态 | ❌ 否 | 当前视角、显示选项 |

**文件特点：**
- 扩展名：.sfa（SFA Files）
- 兼容性：向下兼容旧版本

> **📸 截图建议 3-2：项目文件操作 - 显示打开和保存项目的对话框**  
> **对应文件名称：** `ch03_02_项目文件操作.png`  
> **重要性：** ⭐⭐⭐ 必须

### 3.2.2 保存项目

**保存步骤：**
1. 点击"文件"菜单
2. 点击"储存专案"按钮
3. 选择保存位置并输入文件名
4. 点击"保存"按钮

> **💡 保存建议：**  
> - 定期保存项目，避免数据丢失
> - 使用有意义的文件名（如：零件名_日期）
> - 重要项目建议保存多个版本备份

### 3.2.3 打开项目

**打开步骤：**
1. 点击"文件"菜单
2. 点击"汇入专案"按钮
3. 选择 .sfa 项目文件
4. 点击"打开"按钮或雙擊選中檔案
5. 软件自动加载所有数据

**加载完成后的显示：**
- 所有零件显示在对象树"零件"节点
- 所有路径显示在对象树"路径"节点
- 视图区显示完整的模型和路径
- 所有参数恢复到保存时的状态

## 3.3 完整工作流程

### 3.3.1 标准流程图

```
开始新项目
    ↓
导入 3D 模型（STEP/IGES）
    ↓
生成加工路径
    ↓
加工路徑編程
    ↓
保存项目文件（.sfa）
    ↓
输出 NC 程序
    ↓
[可选] 重新打开项目继续编辑
```
