## ?? 常見錯誤範例

### ? 錯誤寫法（台灣用語）

```markdown
1. 使用滑鼠點選軟體的選單
2. 將檔案匯入到資料夾中
3. 在對話方塊中進行設定
4. 雷射切割的程式參數
5. 儲存專案檔到硬碟
```

### ? 正確寫法（大陸用語）

```markdown
1. 使用鼠標點擊軟件的菜單
2. 將文件導入到文件夾中
3. 在對話框中進行設置
4. 激光切割的程序參數
5. 保存項目文件到硬盤
```

## ?? 特殊處理規則

### 保持原樣的詞彙

以下詞彙**不需要翻譯**，保持原樣：

1. **軟件名稱**：MyCAM
2. **技術術語**：CAD、CAM、NC、VNC、STL、STEP、IGES
3. **技術品牌**：OpenCascade、TightVNC、.NET Framework
4. **代碼相關**：
   - 類別名：`StartupForm`、`DataManager`
   - 方法名：`MethodName()`
   - 變量名：`m_Viewer`
5. **文件擴展名**：.sfa、.step、.iges、.stl、.png

### 混合使用示例

```markdown
? 正確：使用 MyCAM 軟件的 CAD 功能導入 STEP 文件
? 正確：在 `StartupForm` 類中調用 `LoadFile()` 方法
? 正確：將文件保存為 .sfa 格式
```

## ?? 常見陷阱

### 容易混淆的詞彙

| 詞彙 | 台灣用語 | 大陸用語 | 注意事項 |
|------|---------|---------|---------|
| 程式/程序 | 程式 | 程序 | "程序"在大陸可指軟件程序或操作步驟 |
| 文件/檔案 | 檔案 | 文件 | "文件"在大陸可指電子文件或紙質文件 |
| 複製/拷貝 | 複製 | 複製/拷貝 | 兩者在大陸都可用 |

### 需要注意的標點符號

| 台灣 | 大陸 | 說明 |
|------|------|------|
| 「」 | "" | 引號使用雙引號 |
| 『』 | '' | 內層引號使用單引號 |
| ，。 | ，。 | 逗號和句號相同 |
