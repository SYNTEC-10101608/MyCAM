## 1.1 軟體簡介

**MyCAM** 是一套專為五軸切割加工設計的 CAD/CAM 整合軟體系統。本軟體整合了電腦輔助設計（CAD）與電腦輔助製造（CAM）功能，提供從 3D 模型匯入、路徑規劃、刀具姿態調整到 NC 程式輸出的完整解決方案。

軟體採用 OpenCascade 幾何核心，提供強大的 3D 幾何運算能力，讓使用者能夠精確地進行加工路徑設計與刀具姿態調整。

## 1.2 軟體特色與適用情境

### 主要功能

MyCAM 軟體主要提供以下核心功能：

**1. CAD 功能**
- 3D 模型匯入與處理
- 模型縫補與特徵建立
- 座標系統轉換與對齊

**2. CAM 功能**
- 輪廓路徑生成（支援異形蓋件與管拉件）
- 刀具姿態設定與調整
- 刀具姿態調適（支援五軸運動）
- 路徑編輯與排序
- NC 程式輸出

**3. 模擬功能**
- 五軸加工模擬
- 碰撞檢測
- 加工過程可視化

**4. 遠端控制**
- VNC 遠端連線
- 檔案即時傳輸

### 適用情境

**MyCAM** 適用於以下加工類型：

- **異形蓋件**：汽車覆蓋件、家電外殼
- **三曲面類件**：航空機翼件的自動化路徑生成
- **管拉件**：管類零件的專業路徑規劃
- **複雜加工**：需要精確刀具姿態控制的五軸運動加工
- **遠端協作**：支援遠端監控與路徑規劃

## 1.3 系統需求

### 硬體需求

| 項目 | 最低需求 | 建議配置 |
|------|----------|----------|
| 處理器 | Intel Core i5 或同等級 | Intel Core i7 或更高 |
| 記憶體 | 8 GB RAM | 16 GB RAM 或更高 |
| 顯示卡 | 支援 OpenGL 3.3 以上 | 專業顯示卡，支援 OpenGL 4.5 |
| 硬碟空間 | 5 GB 可用空間 | 10 GB 以上（含專案檔空間） |
| 螢幕解析度 | 1920 × 1080 | 2560 × 1440 或更高 |
| 網路 | 選配（用於 VNC 連線） | 100 Mbps 以上有線網路 |

### 軟體需求

| 項目 | 需求 |
|------|------|
| 作業系統 | Windows 10 或 Windows 11（64 位元） |
| .NET Framework | .NET Framework 4.8 或更高版本 |
| OpenCascade | 已整合至軟體（軟體內建） |
| VNC Viewer | TightVNC 或相容軟體（用於遠端連線功能） |

### 其他需求

- **授權認證**：軟體啟動時需通過授權驗證
- **VNC 伺服器**：需與加工機台位於同一區域網路（用於遠端控制功能）
- **機台設定檔**：`MachineData.mac` 檔案需正確配置

## 1.4 授權與啟動流程

### 授權驗證

MyCAM 採用授權驗證機制，軟體啟動時會自動進行授權檢查。

**授權檢查流程：**

1. 軟體啟動時會檢查授權狀態
2. 若授權檢查失敗，系統將顯示錯誤訊息並自動關閉
3. 授權檢查成功後，軟體正常啟動

> **?? 注意事項：**
> - 授權流程由 `FALicenseChecker.LicenseChecker.IsLicenseActivated()` 處理
> - Debug 模式下會略過授權檢查（提供開發測試用）
> - 若出現授權錯誤，請聯絡系統管理員或軟體供應商

### 軟體啟動

**步驟 1：檢查前置條件**
- 確認 .NET Framework 4.8 已安裝
- 確認顯示卡驅動程式已更新

**步驟 2：啟動軟體**
- 雙擊執行 `MyCAM.exe`
- 系統將自動進行授權驗證

**步驟 3：初始化檢查**

軟體啟動後會進行以下初始化：

1. **Viewer 初始化**
   - 若初始化失敗，會顯示訊息：「初始化 Viewer 錯誤，請檢查顯示卡是否已安裝驅動」
   - 請確認 OpenCascade 相關套件已正確安裝

2. **機台數據載入**
   - 系統會嘗試讀取 `MachineData.mac` 機台設定檔
   - 若讀取失敗，系統會使用預設機台參數，並顯示提示訊息

3. **機台模型載入**
   - 根據機台類型（Table / Mix / Spindle）載入對應的 3D 模型檔案
   - 模型檔案位置：軟體目錄/機台類型（Table / Mix / Spindle）
