# 第一章 軟體介紹

## 1.1 軟體概述

**MyCAM** 是一套專為五軸切割加工設計的 CAD/CAM 整合軟體系統。本軟體結合了電腦輔助設計（CAD）與電腦輔助製造（CAM）功能，提供從 3D 模型匯入、路徑規劃、刀具姿態調整到 NC 程式輸出的完整解決方案。

軟體採用 OpenCascade 技術核心，提供強大的 3D 視覺化能力，讓使用者能直觀地進行加工路徑設計與模擬驗證。

## 1.2 軟體用途與應用場景

### 主要功能

MyCAM 軟體主要提供以下核心功能：

**1. CAD 功能**
- 3D 模型匯入與處理
- 模型修補與特徵編輯
- 座標系統轉換與對齊

**2. CAM 功能**
- 智慧路徑生成（支援異形蓋件、管拉件）
- 刀具路徑編輯與優化
- 刀具姿態調整（支援五軸聯動）
- 路徑排序與優化
- NC 程式轉出

**3. 模擬功能**
- 五軸加工模擬
- 碰撞檢測
- 加工過程可視化

**4. 遠端控制**
- VNC 遠端連線
- 控制器即時監控

### 應用場景

**MyCAM** 適用於以下加工場景：

- **航空航太零件**：複雜曲面的精密切割
- **汽車覆蓋件**：異形蓋件的自動化路徑生成
- **管材加工**：管拉件的專業路徑規劃
- **模具加工**：複雜立體曲面的五軸聯動加工
- **金屬板材切割**：雷射或水刀切割路徑規劃

## 1.3 系統需求

### 硬體需求

| 項目 | 最低需求 | 建議配置 |
|------|----------|----------|
| 處理器 | Intel Core i5 或同等級 | Intel Core i7 或更高 |
| 記憶體 | 8 GB RAM | 16 GB RAM 或更高 |
| 顯示卡 | 支援 OpenGL 3.3 以上 | 獨立顯示卡，支援 OpenGL 4.5 |
| 硬碟空間 | 5 GB 可用空間 | 10 GB 以上（含專案檔案空間） |
| 螢幕解析度 | 1920 × 1080 | 2560 × 1440 或更高 |
| 網路 | 選配（用於 VNC 連線） | 100 Mbps 以上區域網路 |

### 軟體需求

| 項目 | 需求 |
|------|------|
| 作業系統 | Windows 10 或 Windows 11（64位元） |
| .NET Framework | .NET Framework 4.8 或更高版本 |
| OpenCascade | 需已安裝相關元件（軟體內建） |
| VNC Viewer | TightVNC 或相容軟體（用於遠端連線功能） |

### 其他需求

- **授權認證**：軟體啟動時需通過授權驗證
- **VNC 執行檔**：需與程式執行檔放置於同一資料夾（用於遠端控制功能）
- **機構設定檔**：`MachineData.mac` 檔案需正確配置

## 1.4 授權與啟動說明

### 授權機制

MyCAM 採用授權保護機制，軟體啟動時會自動進行授權驗證。

**授權驗證流程：**

1. 軟體啟動時會檢查授權狀態
2. 若授權驗證失敗，系統將顯示錯誤訊息並自動關閉
3. 授權驗證成功後，軟體正常啟動

> **?? 注意事項：**
> - 授權流程由 `FALicenseChecker.LicenseChecker.IsLicenseActivated()` 執行
> - Debug 模式下會略過授權檢查（僅供開發測試）
> - 若出現授權錯誤，請聯繫系統管理員或軟體供應商

### 首次啟動

**步驟 1：檢查環境**
- 確認 .NET Framework 4.8 已安裝
- 確認顯示卡驅動程式已更新

**步驟 2：啟動軟體**
- 雙擊執行 `MyCAM.exe`
- 系統將自動進行授權驗證

**步驟 3：初始化檢查**

軟體啟動後會進行以下初始化：

1. **Viewer 初始化**
   - 若初始化失敗，會顯示訊息：「初始化 Viewer 錯誤，請檢查套件是否安裝齊全」
   - 請確認 OpenCascade 相關元件已正確安裝

2. **機構資料載入**
   - 系統會嘗試讀取 `MachineData.mac` 機構設定檔
   - 若讀取失敗，系統會使用預設機構參數，並顯示警告訊息

3. **機構網格載入**
   - 根據機構類型（Table / Mix / Spindle）載入對應的 3D 網格檔案
   - 網格檔案應放置於對應資料夾（Table / Mix / Spindle）
