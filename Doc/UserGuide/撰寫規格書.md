### 1.3 頁數與截圖控制規範

#### 1.3.1 頁數控制原則

| 項目 | 規範 | 說明 |
|------|------|------|
| **每章頁數** | 8-10 頁 | 以 A4 紙張、12pt 字體為基準 |
| **內容精簡** | 必須 | 聚焦核心功能，避免過度詳細 |
| **優先級排序** | 建議 | 先說明最重要的功能 |
| **段落控制** | 每段不超過 10 行 | 保持可讀性 |
| **小節控制** | 合理分配 | 避免單一小節過長 |

**頁數估算方法：**
- 標題（H2）：約 1 行
- 段落文字：每頁約 40-50 行
- 表格：視欄位數量，約 10-20 行
- 截圖（含說明）：約 15-20 行
- 程式碼區塊：視長度，約 10-30 行

**內容精簡技巧：**
1. 合併相似功能的說明
2. 使用表格代替冗長文字
3. 只保留最核心的操作步驟
4. 將次要內容移至附錄

#### 1.3.2 截圖數量控制

| 項目 | 規範 | 說明 |
|------|------|------|
| **每章截圖上限** | 最多 5 張 | 嚴格限制 |
| **必須截圖** | ??? | 核心功能（1-2 張） |
| **建議截圖** | ?? | 重要功能（2-3 張） |
| **可選截圖** | ? | 補充說明（0-1 張） |
| **截圖命名** | 必須 | 每個截圖須有名稱 |

**截圖選擇原則：**
1. **優先選擇最具代表性的畫面**
   - 能展示多個功能的綜合畫面優於單一功能
   - 完整介面優於區域特寫
   
2. **避免重複性截圖**
   - 相似的對話框只截圖一個代表
   - 不同步驟若畫面相似，可合併說明

3. **重要性分級**
   - ??? 必須：軟體主畫面、核心功能操作
   - ?? 建議：重要對話框、關鍵設定
   - ? 可選：補充說明、進階功能

## 2. 文件結構規格

### 2.1 章節架構標準

```markdown
# 第 N 章 章節名稱

## N.1 第一節

### N.1.1 子節（可選）

#### N.1.1.1 細節說明（極少使用）

## N.2 第二節

...
```

### 2.2 標題層級使用規則

| 層級 | Markdown 語法 | 用途 | 範例 |
|------|---------------|------|------|
| **H1** | `#` | 章節標題 | `# 第一章 軟體介紹` |
| **H2** | `##` | 主要小節 | `## 1.1 軟體概述` |
| **H3** | `###` | 次要小節 | `### 主要功能` |
| **H4** | `####` | 三級小節 | `#### 操作步驟詳解` |
| **H5** | `#####` | 極少使用 | 特殊情況下的細分 |

### 2.3 標題編號規則

**章節編號格式：**
- 第一層：`第 N 章`（中文數字或阿拉伯數字）
- 第二層：`N.1`, `N.2`, `N.3`...（阿拉伯數字）
- 第三層：`N.1.1`, `N.1.2`...（阿拉伯數字）
- 第四層：不強制編號，可使用文字標題

**範例：**
```markdown
# 第四章 CAD 功能

## 4.1 模型處理

### 4.1.1 模型縫補

#### 操作步驟
```

## 9. 檔案管理規格

### 9.1 檔案命名規範

#### 9.1.1 章節檔案命名

```
第[章節號]章_[章節名稱].md
```

**規則：**
- 使用中文數字或阿拉伯數字（保持一致）
- 使用底線 `_` 分隔
- 副檔名為 `.md`

**範例：**
- ? `第一章_軟體介紹.md`
- ? `第1章_軟體介紹.md`
- ? `第一章-軟體介紹.md` （不使用破折號）
- ? `chapter1_introduction.md` （不使用英文）

#### 9.1.2 附錄檔案命名

```
附錄[編號]_[附錄名稱].md
```

**範例：**
- `附錄A_快捷鍵列表.md`
- `附錄B_機構設定檔格式.md`

#### 9.1.3 截圖檔案命名

**命名格式：**
```
ch[章節號]_[序號]_[截圖名稱].png
```

**命名規則：**
- 章節號：兩位數字（如 01, 02）
- 序號：兩位數字（01-05，因為每章最多 5 張）
- 截圖名稱：簡短、具描述性的中文名稱（2-6 字）
- 副檔名：`.png`（優先）或 `.jpg`

**正確範例：**
- ? `ch01_01_主畫面總覽.png`
- ? `ch02_03_CAM工具列.png`
- ? `ch03_05_路徑編輯對話框.png`
- ? `ch04_02_模型縫補.png`

**錯誤範例：**
- ? `ch1_1_main.png` （章節號和序號應為兩位數）
- ? `screenshot_2024.png` （缺少章節號和截圖名稱）
- ? `ch02_03_this_is_a_very_long_name.png` （名稱過長）
